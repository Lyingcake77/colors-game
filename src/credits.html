Credits

This game is originally based on a game that I wrote back in 1991 while in college. The original game was written in Turbo Pascal with a small amount of inline 80x86 Assembler.

You can still download the original game, thanks to the Internet Archive. It's a whopping 54K download!

This new game would not be possible without:

  • Phaser (Community Edition) - A cool, open-source Javascript game development library.
  • SoundImage.org - An awesome library of free, royal-free music, sounds effects, and artwork.
  • Affinity Designer - Powerful, yet reasonable-priced, vector and pixel drawing software.
  • Google Fonts - Fast, beautiful, open web-based typography.
